Design System

Prism Memory System for AI Brain

A calm, source-aware UI language inspired by the faceted brain logo. The system uses near-white work surfaces and deep ink structure, with vivid prism colors reserved for source type, capture quality, Ask scope, repair state, and focused moments.

01

Brand Foundation

AI Brain is a private personal knowledge system. The interface should feel trustworthy for long reading sessions, but still carry the energy of intelligence, memory, and synthesis.

Private

Use sober surfaces, clear provenance, and readable structure.

Faceted

Represent different source types and quality levels as named facets.

Vivid When Useful

Reserve saturated color for meaning, state, and orientation.

09

Accessibility and Usage Rules

  • 1Never rely on color alone. Pair color with text labels and icons where useful.
  • 2Keep saturated colors below 15 percent of normal operational screens.
  • 3Use Citrine only for highlights, never long body text.
  • 4Use Ink text on pale tinted backgrounds for reliable contrast.
  • 5Web can support denser rows, right rails, and bulk actions.
  • 6Android should favor bottom nav, FAB capture, compact filters, and sheets.
  • 7Capture result cards must show status, source platform, captured via, quality, and next action.
  • 8Ask answers should stay sober: citations, scope, and source quality matter more than visual flourish.
