# Wiki Information Architecture Plan

## Reading paths

### Fast orientation

1. Home
2. AI Agent Start Here
3. Product Overview
4. Feature Catalog
5. Architecture Overview

### Change implementation

1. Feature Catalog status row
2. Relevant detailed feature page
3. Feature Architecture
4. Repository Map and Data Model
5. APIs and Integrations
6. Testing/operations and known limitations

### Historical research

1. Ideas and Exploration
2. Feature Council Research
3. Dated v2 proposal, then v1/review history when needed

The existing Feature Council pages remain reachable under a Historical Research section. Compatibility pages or links handle any renamed living pages; no valuable history is silently removed.

## Navigation design

The sidebar uses five groups: Start Here, Product and Features, Architecture and Data, Development and Operations, and Reference/History. The Home page provides the same primary paths. `_Footer.md` records the verification convention and contribution link.
