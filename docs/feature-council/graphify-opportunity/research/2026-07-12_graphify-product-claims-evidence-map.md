# Graphify product claims — evidence map

Evidence baseline: Graphify `v8` / `v0.9.13`, commit [`eec7a0183847cbdc8a87d92b233759a5204b89fe`](https://github.com/Graphify-Labs/graphify/tree/eec7a0183847cbdc8a87d92b233759a5204b89fe). Official website pages were accessed 2026-07-12. This map evaluates claim support; it does not make an adoption recommendation.

## Status key

- **Supported**: source and/or focused tests match the material meaning.
- **Supported with qualification**: a real capability exists, but wording omits material conditions or overstates scope.
- **Contradicted/currently stale**: current source differs from the claim.
- **Official claim only**: official source makes the claim, but examined evidence does not independently establish the outcome/number.
- **Unknown**: insufficient evidence.

## Highest-impact discrepancies for later synthesis

1. **Query semantics:** marketing’s “plain-English/path/answer” language obscures a lexical seed + bounded traversal implementation. This is the most important product-fit boundary.
2. **Provenance semantics:** EXTRACTED is not simply “AST,” and INFERRED is not simply “model”; local resolvers also infer.
3. **Explanation semantics:** `explain` is node-neighborhood inspection, not per-edge derivation.
4. **Install behavior:** the website says one install registers everywhere; current code/README defaults to Claude and requires named platform installs.
5. **MCP count:** website says 8; source has 10 tools plus six resources.
6. **Privacy/logging drift:** current source logging is opt-in while README still says default-on; “all on-device” must be scoped to code/local backends.
7. **Community algorithm/labels:** Leiden has a Louvain fallback; optional LLM labels and update-label bugs complicate “LLM-free labels.”
8. **Benchmark evidence:** methodology is described, but the referenced harness/results are not in the examined repository, so the headline numbers remain self-reported.
9. **Headline artifacts:** HTML is conditional and capped; the pipeline creates more than three files/sidecars in practice.
