## Capability confidence summary

### High confidence, source + focused tests

- AST/local Markdown extraction and many cross-file relations.
- Node-link graph serialization with confidence/source metadata.
- Lexical seed + BFS/DFS query behavior and undirected shortest path.
- Community detection/fallback, degree-ranked hubs, report generation.
- HTML/GraphML/Obsidian/wiki exports.
- Incremental cache/watch mechanisms at the unit/regression-test level.
- Skill/config installation paths and MCP tool registration.

### Medium confidence, implemented but environment/integration dependent

- Semantic document/image extraction across every backend.
- faster-whisper and Google Workspace conversion in real environments.
- Live PostgreSQL introspection across permissions/schema complexity.
- Neo4j/FalkorDB performance and idempotence at production scale.
- End-to-end behavior in every advertised assistant host.

### Claims, not established capabilities

- Token/cost savings in AI Brain.
- Better recall, retention, onboarding speed, or architecture understanding for AI Brain users.
- Calibrated trustworthiness of numeric confidence scores.
- Stable semantic edges/community names across models and repeated runs.
- Suitability as a replacement for AI Brain’s current search or memory model.
