| R-023 | Raw identifiers/hashes or private URL parameters leak remotely | Medium/High | Allowlist mapper, full connection-scoped HMAC markers, URL default-deny, malformed-provenance fail-closed behavior | Local fixture tests pass; repeat against final mapper if implementation is later authorized |
