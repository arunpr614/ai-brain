## 0. User-locked constraints (confirmed 2026-05-12)

These are non-negotiable and shape every spike:

1. **Always-on architecture required.** Mac sleeping = Brain unreachable is the pain point we're solving.
2. **No home server, no Mac Mini.** Rules out RPi / used mini PC / Synology / buy-a-Mac-mini. **Cloud VM only.**
3. **AI source is open** — paid API OR hybrid (embeddings local + generation cloud, or vice versa). NOT "keep Ollama on the cloud VM" unless spike proves it's cheaper than API.
4. **Enrichment timing: daily batch is the default.** Immediate enrichment is NOT required. A future "Enrich now" button for per-item manual triggers is the escape hatch.
5. **Cost is a top optimization target.** We want the cheapest cloud + AI combo that meets the reliability + privacy bar.

## 5. Expected cost envelope (pre-spike; will be revised)

Given the constraints and some market knowledge:

| Architecture | Monthly cost | Notes |
|---|---|---|
| **Oracle Free Tier (A1 Ampere) + Claude Haiku API** | $0 compute + $1-5 API | Free VM with 4 ARM cores + 24 GB RAM; too tempting to not evaluate despite signup friction |
| **Hetzner CX22 (2 vCPU, 4 GB) + Claude Haiku** | €4.59 + $1-5 API | Known-reliable; Germany data center |
| **Hetzner CPX11 (2 AMD vCPU, 2 GB) + Gemini 2.5 Flash** | €3.79 + $1-3 API | Cheaper tier; Gemini Flash is known-cheap |
| **Fly.io shared-cpu-1x + gpt-4o-mini** | $1.94 + $1-2 API | Shared CPU; may struggle under enrichment burst |
| **AWS Lightsail 2 GB + Haiku** | $10 + $1-5 API | Comfortable but more expensive |

**Realistic target: $3-10/month total.** The AI API cost is small (your volume is low); the cloud host dominates. If Oracle Free Tier works for your stack, it's effectively free forever.
