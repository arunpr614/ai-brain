# AI Memory Brand Copy Migration

Created: 2026-06-13 22:34 IST

## Required Product Name

The implemented product name is `AI Memory`.

Use `AI Memory` in:

- Login and unlock headings.
- Navigation labels.
- Ask composer and empty states.
- Settings/account surfaces.
- Capture confirmation copy.
- Browser metadata, app title, favicon labels, and Android launcher labels.

## Legacy Name Rule

`AI Brain` is the older project/prototype name.

Allowed uses:

- Historical filenames.
- Adversarial review evidence.
- Exact Magic Patterns source exports where preserving prototype provenance matters.
- Source-reference notes that explicitly identify the old project name.

Disallowed uses:

- Production UI.
- Implementation-facing screen specs.
- User-facing screenshot labels.
- Android app name, launcher name, or login copy.
- Web app title, page title, or favicon title.
- Ask composer labels.
